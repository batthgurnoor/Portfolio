## Portfolio

A responsive portfolio website for, showcasing programming projects, photography work, experience, certifications, and testimonials.

### Features
- **Responsive layout** built with HTML, CSS/SCSS, and Bootstrap
- **Sidebar navigation** with smooth in-page section links
- **Home section** with intro, social links, and resume download
- **About section** with bio, skills, languages, IT skills, and certifications
- **Resume section** with education and experience timeline
- **My Work section** with filterable gallery (Programming / Photography) and project detail modal
- **Testimonials section** with rotating feedback
- **Contact section** with contact form and direct email/phone/location info
- **Dark/light mode toggle** and animated visual effects (meteors, parallax, typed text)

### Tech Stack
- **HTML5**
- **CSS3 / SCSS** (`assets/scss`, compiled to `assets/css/main.css`)
- **JavaScript** (`assets/js/script.js`)
- **Bootstrap 4 & 5**
- **jQuery**
- **Typed.js** (typed text effect on the home header)
- **Atropos** (3D hover/tilt effect for hero image)
- **Shuffle.js** (filtering and layout for the work grid)
- **EmailJS** (for handling contact form submissions via email)

### Project Structure
- **`index.html`**: Main single-page application
- **`assets/css/`**: Compiled styles (`main.css`, source map)
- **`assets/scss/`**: Modular SCSS partials and `main.scss`
- **`assets/js/script.js`**: Site interactions, animations, and component behavior
- **`assets/images/`**: All images (hero, skills icons, works, testimonials, certificates, backgrounds)
- **`assets/files/IT_Resume.pdf`**: Downloadable resume
